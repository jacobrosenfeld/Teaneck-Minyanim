About Calendar Events

Rules-Based: Generated automatically from minyan schedules. Re-materialized weekly.
Imported: From external calendar imports. Can be disabled but not deleted.
Manual: Created manually by admins. Can be edited and deleted.
Materialization Window: Start to End (3 weeks past, 8 weeks future)
