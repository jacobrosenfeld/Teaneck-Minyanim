Notification Management

Manage site-wide notifications for banners and popups

How it works: Each notification has a unique ID for cookie-based tracking. When a user views a notification, it's tracked in their browser cookies. You can set max displays per notification and expiration dates. Banner notifications appear at the top of the homepage, while popup notifications appear in a modal.
