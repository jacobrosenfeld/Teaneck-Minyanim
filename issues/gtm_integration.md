## Create Google Tag Manager Integration

### Overview
We need to create a Google Tag Manager (GTM) integration for both the mobile app and website. This integration will allow users to insert their GTM ID into a settings page.

### Requirements
1. **Mobile App Integration**: Develop the necessary features to accept and use GTM IDs in the mobile application.
2. **Website Integration**: Implement a settings page on the website where users can input their GTM ID.
3. **Testing**: Ensure that the GTM integration works seamlessly on both platforms.

### Acceptance Criteria
- Users can input and save their GTM ID.
- GTM ID is properly utilized in the mobile app and website tracking.
